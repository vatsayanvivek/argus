# cis_1_14 — Ensure Privileged Identity Management (PIM) is in use

!!! note "Summary"
    **Severity:** :material-alert-circle-outline: Medium · **Pillar:** Identity · **Chain role:** ENABLER

## Description

PIM provides just-in-time elevation and reduces standing privileged access. Tenants should have at least some eligible (non-permanent) assignments configured.
