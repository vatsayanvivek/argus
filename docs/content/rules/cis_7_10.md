# cis_7_10 — Only approved VM extensions are installed

!!! note "Summary"
    **Severity:** :material-alert-circle-outline: Medium · **Pillar:** Workload · **Chain role:** AMPLIFIER

## Description

VM extensions execute code with elevated privileges on the host. Unapproved extensions can be used for persistence, privilege escalation, or command execution.
