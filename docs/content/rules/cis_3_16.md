# cis_3_16 — Storage account uses private endpoints

!!! note "Summary"
    **Severity:** :material-alert: High · **Pillar:** Data · **Chain role:** ENABLER

## Description

Storage accounts accessible over the public internet expose data to network-based attacks. Private endpoints restrict access to the virtual network and eliminate public exposure.
