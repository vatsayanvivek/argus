# cis_1_12 — Ensure guest users do not have privileged role assignments

!!! note "Summary"
    **Severity:** :material-alert-octagon: Critical · **Pillar:** Identity · **Chain role:** ANCHOR

## Description

B2B guest users holding privileged roles create cross-tenant trust that is difficult to audit and control. Privileged guests are a top persistence vector.
