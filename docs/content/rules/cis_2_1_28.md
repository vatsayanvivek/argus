# cis_2_1_28 — Defender for Azure SQL Database enabled

!!! note "Summary"
    **Severity:** :material-alert: High · **Pillar:** Data · **Chain role:** ENABLER

## Description

Defender for Azure SQL Database detects SQL injection, anomalous access, and brute-force attacks against Azure SQL databases. Without it, database-layer attacks go undetected.
