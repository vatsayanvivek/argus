# cis_1_13 — Ensure access reviews exist for privileged roles

!!! note "Summary"
    **Severity:** :material-alert-circle-outline: Medium · **Pillar:** Identity · **Chain role:** ENABLER

## Description

Azure AD Access Reviews should be configured to periodically validate continued need for privileged role assignments.
