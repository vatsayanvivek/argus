# cis_1_24 — Custom subscription owner roles are not created

!!! note "Summary"
    **Severity:** :material-alert-circle-outline: Medium · **Pillar:** Identity · **Chain role:** AMPLIFIER

## Description

Custom roles with Owner-equivalent permissions bypass the governance controls applied to built-in Owner. Custom owner roles should not be created; use built-in Owner with PIM instead.
