# cis_1_25 — Role assignments use groups instead of individual users

!!! note "Summary"
    **Severity:** :material-information-outline: Low · **Pillar:** Identity · **Chain role:** AMPLIFIER

## Description

Direct user role assignments create management overhead and increase the risk of orphaned permissions. Group-based assignments enable centralized access governance and simplify auditing.
