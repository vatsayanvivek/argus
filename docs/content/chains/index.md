# Attack chains

ARGUS ships **200 attack chains** — realistic, multi-step attack narratives that correlate individual findings into end-to-end paths. A chain fires only when real findings in your scan match its trigger pattern, so the output reads like an incident write-up, not a checklist.

## Trigger logic reference

| Logic | Meaning |
|---|---|
| `ALL` | Every rule in the trigger set must fire. |
| `ANY_TWO` | At least two rules in the trigger set must fire. |
| `ANCHOR_PLUS_ONE` | The anchor rule must fire AND at least one other trigger. |
